# Landing Page Project

This repository contains a simple static landing page built with HTML and CSS. The page is designed to showcase a clean hero section, brand logo, and feature photos in a modern layout. It is ideal as a lightweight marketing page or visual presentation for a product or service.

## What this is about

- A static homepage layout using `index.html` and `style.css`
- A hero section with a large visual image and branding
- A set of photos displayed as feature cards
- A responsive design approach for easy viewing in a browser

## How to view

Open `index.html` in your web browser to see the landing page live. The page uses assets from the `images/` folder.
